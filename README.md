# Favorize Feedback

This repository is for collecting feedback, bug reports, and feature requests for the **Favorize** app.

## How to Submit Feedback

1. Go to the [Issues](https://github.com/favorize-app/favorize-feedback/issues) tab
2. Click **New Issue**
3. Choose a template:
   - **Bug Report** — something isn't working as expected
   - **Feature Request** — an idea for a new feature or improvement
   - **Question** — general questions about the app
4. Fill in the template and submit

## What Happens Next

Our team reviews all submissions. We'll respond on the issue with updates, questions, or a decision. Please check back for responses.

## Guidelines

- Search existing issues before creating a new one — your topic may already be tracked
- One issue per topic — keep issues focused
- Be specific — include steps to reproduce bugs, or describe exactly what you'd like for feature requests
- Be respectful — constructive feedback helps us build a better app
